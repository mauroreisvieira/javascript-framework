## Get Started

```bash
git clone https://github.com/mauroreisvieira/mddoks
```

Alternatively you can [download](https://codeload.github.com/mauroreisvieira/mddoks/zip/master) this repository.

## About Mddoks

Mddoks is an documentation builder that helps you quickly build documentations.\
It's powered by [Stewed](http://mauroreisvieira.github.io/stewed/#/), allowing you to create custom themes, which makes the process of creating documentation easy.
